Gesture Snake

Instead of keyboard this snake is controlled by your hand movements!

Instructions

To play with Gestures you need MAX32655FTHR board, MAX25405EVKIT and tools required for soldering (+ in Europe you need 3.3V@1A power supply like this one because MAX25405EVKIT contains power supply with US plug).

For playing with gesture sensor you need do following:

  1. Solder pin headers to MAX32655FTHR Board
  2. Connect EVKIT Shield With MAX32655FTHR Board.
  3. Connect EVKIT Sensor Board using ribbon cable to shield and MAX32655FTHR
  4. Plug power supply adapter and connect it to EVKIT Shield Board
  5. Connect USB Cable between your computer and MAX32655FTHR
  6. Download firmware and drag-and-drop downlaoded file to DAPLINK thumbdrive
  7. Click the button and select the device:
  8. Swipe hand from left to right to start the game

Source

Game is open source and open hardware. GitHub project contains source codes of this web game and also sources of the firmware including sensor driver and gesture detection algorithm. Hardware resources related to MAX32655FTHR and MAX25405EVKIT are avalaible at vendor (Analog Devices) website free to download.
